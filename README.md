# מחשבון משיכת אנונה מקצועי - גרסה מעודכנת עם תובנות כלכליות

מחשבון אנונה מתקדם ואינטראקטיבי המבוסס על קובץ Excel מקורי ונוסחאות פיננסיות מדויקות, עם תמיכה מלאה במס ריאלי ונומינלי ושיטות משיכה שונות.

המחשבון מתחיל עם **3 פרמטרים בסיסיים בלבד** כדי לאפשר חישוב מהיר וקל. משתמשים מתקדמים יכולים לפתוח **אפשרויות מתקדמות** לכוונון מדויק. כל הפרמטרים מגיעים עם **ערכי ברירת מחדל אופטימליים** המבוססים על נתונים מקצועיים.

### 🧮 **מחשבון אנונה מדויק לפי הקובץ המקורי**
- **נוסחאות פיננסיות מדויקות** מהקובץ "מחשבון אנונה סופי.xlsx"
- **חישוב ריביות חודשיות אפקטיביות**: `rm = (1+R)^(1/12)-1`

- **מיסוי דיפרנציאלי**: 25% ריאלי מול 15% נומינלי
- **שיטות משיכה**:
  - **ברוטו (לפני מס)**: מגדיר כמה למשוך לפני ניכוי מס - הסכום בפועל יהיה נמוך יותר
  - **נטו (אחרי מס)**: מגדיר כמה לקבל בפועל אחרי מס - המחשבון יחשב כמה למשוך ברוטו
